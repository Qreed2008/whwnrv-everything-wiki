Windows 9x

Windows 9x is a series (and kernel type) of Windows versions that are year-like, but it's the year minus 1900.

(00 - 99 are 1900 - 1999)

Single digit versions are still double number in WHwNRV Everything, ex. 5 is 05, -5 is -05.

Similar to this is Windows '0x, a 2000's version of this.
