1989

  • January 20 - Windows 3.0 build 14 (earlier build) is compiled.
  • February 22 - Windows 3.0 build 14 (later build) is compiled.
  • November 27 - Windows 3.0 build 48 is compiled.

1990

  • January 2 - Windows 3.0 build 55 is compiled.
  • February 23 - Windows 3.0 RC2 is compiled.
  • March 7 - Windows 3.0 RC6 is compiled.
  • May 1 - Windows 3.0 release/RTM build is compiled.
  • May 22 - Windows 3.0 is released.
  • July 2 - Windows 3.0 "Test Drive", a stripped-down Windows 3.0 installation free with some PC World magazines, is compiled.
  • October 31 - Windows 3.00a is released.

1991

  • January 14 - An Intel Inboard 386/PC version of Windows 3.00a is released.
  • November 22 - Windows 3.02 is released.

2014

  • March 3 - Windows 3.0 build 55 is re-leaked from a pre-installed folder originally leaked in 1990.

2015

  • January 13 - Windows 3.0 build 14 (later build) is leaked.

2019

  • September 17 - Windows 3.0 RC2 is leaked to BetaWorld. (separate because downloads aren't on that site)

2020

  • April 1 - Windows 3.0 RC2 is leaked.
  • June 8 - A video of Windows 3.0 build 48 is posted on YouTube.
