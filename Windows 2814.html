Windows 2814

Windows 2814 (codenamed Vaporwave 2000) is a version of Windows released in mid-2000 (even though the copyright line says 199X)

After the release of this version, the dev team branched off to make similar things for later versions of Windows. These later releases are known as Vaporwave OS. <-- this may change, i.e. maybe a stupid idea
